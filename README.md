# dotfiles

Personal dotfiles managed by [chezmoi](https://www.chezmoi.io/), with OS-specific templates for macOS and Linux.

## What's Included

| Config | Target | Notes |
|--------|--------|-------|
| **nvim** | `~/.config/nvim/` | Lazy.nvim, native LSP (`vim.lsp.config`), Telescope, Treesitter, Harpoon, Verilog (hdl_checker) |
| **zsh** | `~/.zshrc` | Template with OS conditionals |
| **tmux** | `~/.config/tmux/tmux.conf` | Catppuccin, vim-tmux-navigator, TPM |
| **starship** | `~/.config/starship.toml` | Template: gradient theme (macOS) / nerd font symbols (Linux) |
| **aerospace** | `~/.config/aerospace/aerospace.toml` | macOS tiling window manager |
| **claude** | `~/.claude/` | Claude Code agents, skills, settings |

## Setup

```bash
# Install chezmoi and apply
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply linrswa

# Install tmux plugins (first run)
# Open tmux, then press prefix + I
```

## OS-Specific Templates

`.zshrc` and `starship.toml` use chezmoi templates to handle platform differences:

- **macOS**: Homebrew plugin paths, `bat`, conda, fzf via `source <(fzf --zsh)`
- **Linux**: System package paths, `batcat`, Ghostty term fix, fzf via `source ~/.fzf.zsh`

## Sync Helper

```bash
./sync.sh
```

Detects new files in chezmoi-managed directories, offers to add them, then runs `chezmoi re-add` to sync target changes back to the source.

## Externally Managed

These are **not** tracked by chezmoi:

- `~/.config/tmux/plugins/` — managed by [TPM](https://github.com/tmux-plugins/tpm)
- `~/.config/nvim/lazy-lock.json` — generated by [lazy.nvim](https://github.com/folke/lazy.nvim)
